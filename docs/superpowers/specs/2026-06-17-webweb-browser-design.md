# WebWeb Browser - 设计文档

## 1. 项目概述

**项目名称：** WebWeb Browser

**项目目标：** 创建一个基于iframe的浏览器标签页模拟系统，用于个人本地日常使用，完全模拟浏览器行为。

**技术栈：** 纯HTML/CSS/JavaScript，多文件分离结构

**使用方式：** 双击`index.html`直接打开，无需服务器或构建工具

---

## 2. 项目结构

```
webweb/
├── index.html          # 主页面
├── css/
│   └── style.css       # 样式文件
├── js/
│   ├── app.js          # 主应用逻辑
│   ├── tab-manager.js  # 标签页管理
│   ├── storage.js      # localStorage管理
│   └── zoom.js         # 缩放控制
└── icons/              # 图标资源（可选）
```

**文件职责：**
- `index.html`：页面结构和布局
- `style.css`：所有样式，包括顶部/左侧布局切换
- `app.js`：初始化和事件协调
- `tab-manager.js`：标签页增删改查逻辑
- `storage.js`：状态保存和恢复
- `zoom.js`：iframe缩放控制

## 3. 界面设计

### 3.1 设计规范

- **颜色：** 纯黑(`#000`)、纯白(`#fff`)、浅灰(`#f5f5f5`)、深灰(`#333`)
- **圆角：** 无，全部直角
- **边框：** 1px solid #ccc
- **字体：** 系统默认字体
- **阴影：** 无

### 3.2 顶部标签栏模式

```
┌─────────────────────────────────────────────────┐
│[+][Tab1 ×][Tab2 ×][Tab3 ×]       [⊞] [−][+]   │
├─────────────────────────────────────────────────┤
│ [                                    ] [→]     │
├─────────────────────────────────────────────────┤
│                                                 │
│              iframe 内容区域                     │
│                                                 │
└─────────────────────────────────────────────────┘
```

### 3.3 左侧标签栏模式

```
┌──────┬──────────────────────────────────────────┐
│ [+]  │ [                                    ] [→]│
│Tab1 ×├──────────────────────────────────────────┤
│Tab2 ×│                                          │
│Tab3 ×│           iframe 内容区域                │
│      │                                          │
│[⊞][−+]│                                         │
└──────┴──────────────────────────────────────────┘
```

### 3.4 交互元素

- `[+]`：新建标签页按钮
- `[TabX ×]`：标签名 + 关闭按钮
- `[⊞]`：布局切换按钮（图标自适应）
- `[− +]`：缩放控制（最小化显示）
- 地址栏：输入URL后回车跳转
- 标签右键菜单：编辑URL、关闭

### 3.5 状态指示

- 活跃标签：白色背景
- 非活跃标签：灰色背景
- 悬停：边框变深

## 4. 数据结构与存储

### 4.1 localStorage存储结构

```json
{
  "webweb_state": {
    "tabs": [
      {
        "id": "tab_1",
        "title": "Google",
        "url": "https://www.google.com",
        "zoom": 1.0
      },
      {
        "id": "tab_2",
        "title": "GitHub",
        "url": "https://github.com",
        "zoom": 0.8
      }
    ],
    "activeTabId": "tab_1",
    "layout": "top",
    "globalZoom": 1.0
  }
}
```

### 4.2 字段说明

- `id`：标签唯一标识（时间戳生成）
- `title`：标签显示标题（从页面自动获取或手动编辑）
- `url`：当前URL
- `zoom`：该标签页的缩放级别（0.1 - 3.0）
- `activeTabId`：当前活跃标签ID
- `layout`：布局模式（"top" 或 "left"）
- `globalZoom`：全局缩放级别

### 4.3 保存时机

- 标签页增删改时
- 切换标签时
- 缩放变化时
- 布局切换时
- 页面关闭前（beforeunload）

## 5. 核心功能逻辑

### 5.1 标签页生命周期

#### 新建标签页
1. 点击`[+]`按钮
2. 生成唯一ID，创建空标签
3. 自动聚焦到地址栏
4. 默认URL：`about:blank`

#### 切换标签页
1. 点击标签切换
2. 隐藏当前iframe，显示目标iframe
3. 更新活跃状态样式
4. 保存到localStorage

#### 关闭标签页
1. 点击`[×]`按钮
2. 移除iframe和标签元素
3. 如果关闭的是活跃标签，自动切换到相邻标签
4. 如果只剩一个标签，禁用关闭按钮

#### 编辑URL
1. 点击标签右键菜单"编辑URL"
2. 或双击标签进入编辑模式
3. 修改后回车确认，更新iframe src

### 5.2 缩放控制

- `[−]`：缩小（步长0.1，最小0.3）
- `[+]`：放大（步长0.1，最大3.0）
- 显示当前缩放比例
- 缩放应用到iframe的`transform: scale()`属性
- 每个标签页独立缩放级别

### 5.3 布局切换

- 点击`[⊞]`按钮
- 顶部↔左侧切换
- 动画过渡（可选，保持简洁）
- 保存布局偏好

## 6. 错误处理与边界情况

### 6.1 iframe加载失败

- 显示友好的错误提示（黑色背景白字）
- 提供重试按钮
- 常见错误：跨域限制、网络问题、无效URL

### 6.2 无效URL处理

- 自动添加`https://`前缀（如果缺失）
- 验证URL格式
- 无效URL显示错误页面

### 6.3 localStorage限制

- 存储满时的降级处理（提示用户清理）
- 数据损坏时的恢复机制（重置为默认状态）
- 首次使用时的初始化

### 6.4 性能优化

- 非活跃标签的iframe延迟加载
- 标签页过多时的滚动支持（横向/纵向）
- 内存管理：关闭标签时清理iframe引用

### 6.5 浏览器兼容性

- 现代浏览器（Chrome 80+, Firefox 75+, Safari 13+, Edge 80+）
- 不支持IE
- 使用标准ES6+语法

## 7. 完全浏览器行为模拟

### 7.1 iframe配置

- 不使用`sandbox`属性
- 允许所有权限（摄像头、麦克风、全屏等）
- 允许弹出窗口
- 允许表单提交
- 允许脚本执行
- 允许同源和跨域访问

### 7.2 完全浏览器行为

- 支持所有网页功能（登录、购物、视频播放等）
- 支持JavaScript弹窗（alert、confirm、prompt）
- 支持文件上传下载
- 支持地理位置（如果用户允许）
- 支持通知权限

### 7.3 注意事项

- 某些网站可能通过`X-Frame-Options`或`CSP`阻止iframe嵌入
- 这是网站自身的限制，无法绕过
- 对于这类网站，显示友好提示

### 7.4 隐私考虑

- 所有数据存储在本地localStorage
- 不收集任何用户数据
- 不发送任何网络请求（除了用户访问的网站）

## 8. 技术实现细节

### 8.1 HTML结构

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>WebWeb Browser</title>
  <link rel="stylesheet" href="css/style.css">
</head>
<body>
  <div id="browser">
    <div id="tab-bar">
      <button id="new-tab">+</button>
      <div id="tabs-container"></div>
      <div id="controls">
        <button id="layout-toggle">⊞</button>
        <div id="zoom-controls">
          <button id="zoom-out">−</button>
          <span id="zoom-level">100%</span>
          <button id="zoom-in">+</button>
        </div>
      </div>
    </div>
    <div id="address-bar">
      <input type="text" id="url-input" placeholder="输入URL...">
      <button id="go-btn">→</button>
    </div>
    <div id="content-area">
      <!-- iframes动态插入 -->
    </div>
  </div>
  <script src="js/app.js"></script>
</body>
</html>
```

### 8.2 CSS关键样式

- 使用CSS变量管理颜色
- Flexbox布局实现顶部/左侧切换
- 响应式设计适配不同屏幕
- 过渡动画保持简洁

### 8.3 JavaScript模块

- `storage.js`：封装localStorage操作
- `tab-manager.js`：标签页CRUD操作
- `zoom.js`：缩放逻辑
- `app.js`：初始化和事件绑定

### 8.4 事件处理

- 点击、双击、右键菜单
- 键盘快捷键（Ctrl+T新建，Ctrl+W关闭）
- 窗口大小变化
- 页面加载和卸载
